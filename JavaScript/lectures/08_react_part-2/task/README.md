# React Part 2

## Перед выполнением:
- Закончить задание `07_react_part-1`.
- Дождаться `approval` и `merge` вашего PR. (Пингуйте менторов, когда доделаете таску)
- Cкопировать содержимое папки с заданием `07_react_part-1` к себе в workspace.
- Подключить `eslint` с конфигурацией `airbnb`.

## Выполнение задания:
Написать чат на реакте. Базовый локальны чат у вас уже есть. Ваша задача его улучшить и заполнить реальными данными.

Требования к чату:
- Зарегестировать своё [телеграм приложение](https://core.telegram.org/api/obtaining_api_id).
- Подключить чат к `telegam api`. Реккомендуем пакет [telegram-api-js](https://www.npmjs.com/package/telegram-api-js). Запросы к api выполняйте в `componentDidMount`.
- Добавить окно входа в аккаунт (2 поля: телефон и код авторизации).
- Все данные должны быть получены с `telegram api`. (сообщения, чатч, индикаторы и т.д.)
- Добавить в каждый компонент `prop-types`.
- Добавить команду `npm run lint`, которая должна запускать `eslint`.
- По максимуму использовать функциональные компоненты.

## Advanced
Использовать контекст (`React.createContext`) для передачи **данных** чата в дочерние компоненты.

## Для тех кто кодит по ночам вместо сна
Переписать стили на `styled-components`.
